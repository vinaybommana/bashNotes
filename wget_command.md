# some of the use cases
* if I want to download a file of a particular format recursively in the page.
```
wget -A "*.<format>" -r "<url>"
# for example
wget -A "*.cbr" -r "<url>"
```

* if we want to resume a file
```
wget -A "*.cbr" -c -r "<url>"
# if we don't recursive folders
wget -nd "<url>"
```
